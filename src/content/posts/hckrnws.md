## Description

A Hacker News UI built with [Sveltekit](https://kit.svelte.dev), [Tailwind](https://tailwindcss.com), and [DaisyUI](https://daisyui.com).

## Features

- Top Stories
- New Stories
- Best Stories
- Ask Stories
- Show Stories
- Job Stories
- View Comments
- View Story Content
- View User Profile (WIP)

## Getting Started

```bash
git clone https://github.com/engageintellect/hckrnws.git
```

```bash
cd hckrnws
```

```bash
pnpm install
```

```bash
pnpm run dev
```
